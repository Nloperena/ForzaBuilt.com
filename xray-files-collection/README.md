# X-Ray Files Collection

This folder contains all X-ray related files from the ForzaBuilt website project.

## Structure

### `/components`
All React TypeScript component files for X-ray functionality:
- `ImageOverlay.tsx` - Main overlay component for SVG interactions
- `IndustryXRaySelector.tsx` - Industry-specific X-ray selector
- `IndustryXRaySections.tsx` - X-ray sections wrapper
- Plus various other X-ray explorer and overlay components

### `/svg-overlays`
SVG overlay files organized by industry:
- **transportation/** - RV Bus and Trailer exploded graphics
- **marine/** - Marine boat and pontoon overlays
- **construction/** - Construction house and commercial overlays
- **insulation/** - House and pipe exploded graphics
- **composites/** - Composite turbine overlays
- **industrial/** - Industrial manufacturing overlays

### `/xray-assets`
Complete X-ray asset structure with metadata:
- Pre/post X-ray images
- Overlay SVG files
- Metadata JSON files
- Organized by industry and X-ray variant

## Usage

This collection can be zipped and archived or used as a reference for X-ray component implementations.
